How Does Summaryt Work?

Summaryt is an attempt to create a text summarizer that takes audio file as an input and converts the contents of the audio file into a summarised text for you to read and enjoy.

1. Capturing the audio file

The first step in our process is to capture the audio file that is to be summarised. The website currently supports only upload of audio files.

2. Noise Removal and Pre-processing

This is a preprocessing step used to remove noise from the noise to increase the accuracy of the text summarisation module. The preprocessing involves removal of background noise to obtain clean audio.

3. Speech to text conversion

Once the audio is cleaned and pre processed, it is sent for conversion into text.This step converts the clean audio speech into text that will be used for text summarisation.

4. Text Preprocessing

This stage is responsible for cleaning up the text recieved from the speech to text conversion phase. It involves removing slang words and non native words from the text.

5. Text Summarisation

Once the text is preprocessed it is then summarised using natural language processing tools.

6. Rearrangement and export

Once the text is summarised, it is then presented to the user.The user may download the text as pdf for future use.
